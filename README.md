# ¿De que trata la app-web?

Nuestra app web es una aplicacion pensada para perros, tanto para adopcion como rescate, 
una aplicacion donde los usurios puedan entrar y interactuar con el sistema y ver perritos 
que necesitan ayuda.

# ¿Que lenguaje de desarrollo usa?

El lenguaje en el que se maneja es HTML, CSS, JAVA SCRIPT.
